### `Travel Guru`

Technology Used:- `React.js`  `Bootstrap`  `Firebase`  `React-Router`


•	A travel website for online booking service.

•	Select your desire location and see brief info

•	Traveler can select hotel of their choice.

•	After booking you can see google map location and available hotels.
